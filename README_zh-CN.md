# 像素角色生成器

## 项目概述

将角色图片转换为包含8个视角的游戏像素画精灵图。非常适合游戏开发者、艺术家和动画师创建2D游戏素材或角色参考图。

## 模块功能

### 8方向像素角色生成（子流程）

主工作流程，协调整个转换过程。接收角色和可选参考图片的远程URL，输出完整的8方向精灵图。

**输入：**
- `remote_url`（必填）：角色图片的远程URL（http/https）
- `input`（可选）：像素画风格参考图片的远程URL

**输出：**
- `images`：生成的精灵图图像路径数组

### 输入与提示构建器

将角色和参考图片与像素画生成指令结合。验证输入是否为远程URL。

**输入：**
- `character_image_url`（必填）：角色图片的远程URL
- `reference_image_url`（可选）：风格参考图片的远程URL
- `user_prompt`（可选）：额外的自定义文本

**输出：**
- `final_prompt`：完整的生成指令
- `image_urls`：用于处理的图片URL数组

### URL格式转换器

在不同模块格式之间转换图片URL数组，过滤无效值。

## 模块组合建议

**基础流程：**
上传角色图片 → 8方向像素角色生成

**增强流程：**
上传角色图片 + 上传参考图片 → 8方向像素角色生成

**自定义流程：**
上传图片 → 输入与提示构建器（添加自定义提示）→ 生成像素画 → 生成精灵图

## 基本使用

1. 使用文件上传模块（cloudflare-r2 或 upload-to-cloud）上传角色图片
2. 可选上传参考图片以指导风格
3. 将远程URL连接到8方向像素角色生成子流程
4. 运行工作流生成8方向精灵图

## 示例

**游戏开发：** 将概念设计图转换为精灵图，适用于俯视角或等距视角游戏中角色可向任意方向移动的场景。

**角色艺术：** 将现代角色设计转换为复古16位像素画风格，用于头像或社交媒体内容。

**动画参考：** 生成展示所有8个视角的角色参考图，确保动画制作的一致性。

## 获得最佳效果的小贴士

- **角色图片** 在角色清晰可见、背景简单或透明时效果最好
- **参考图片** 应展示你欣赏的像素画风格——也许是你喜欢的某款游戏
- **自定义提示** 让你添加具体要求，比如"让头发更尖"或"加一件披风"
- **风格一致** 在多个角色之间使用相同的参考图片可以保持统一风格

## 你将得到什么

最终输出是一张包含所有8个角色视角的图片，按网格排列。这种格式：

- 可直接导入游戏引擎
- 适合动画软件使用
- 如有需要可轻松拆分为单独的帧
- 完美作为角色参考图
